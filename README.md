# 🛒 Electronics Shop Website

A modern Electronics Shop website built using **HTML5** and **CSS3**. This project showcases a clean e-commerce homepage with product categories, offers, and a professional footer.

## ✨ Features

- 🏠 Responsive Homepage
- 📱 Electronics Product Categories
- 🎁 Deals & Offers Section
- 🛍️ Shopping Layout
- 🔍 Search Bar
- 📋 Navigation Bar
- 📦 Product Cards
- 📄 Professional Footer
- 💻 Clean and Simple UI

## 🛠️ Technologies Used

- HTML5
- CSS3
